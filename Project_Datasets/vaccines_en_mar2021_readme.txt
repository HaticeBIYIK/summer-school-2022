TWITTER VACCINE ENGLISH MARCH 2021 DATASET
Collected by Yelena Mejova <yelena.mejova@gmail.com>

Tweets were collected using the Twitter Streaming API with keywords related to vaccination (in English). The collection is constrained to tweets labeled by Twitter as being in English (en) language. For each day, data has a random sample of up to 10,000 tweets. It spans March 2021, capturing the suspensions of AstraZeneca vaccine https://en.wikipedia.org/wiki/Oxford%E2%80%93AstraZeneca_COVID-19_vaccine#Suspensions .
